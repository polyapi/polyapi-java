## Description

Poly API.

## Installation

```bash
$ yarn install
```

## Running the app
This will run the server on port 8000 by default.

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Endpoints
`POST /teach`\ - prepares Poly API function
`POST /teach/:functionId` - updated Poly API function with response and/or metadata
